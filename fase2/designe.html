FlexBox

Flexbox viene de “Flexible Box Layout“, que se puede traducir como “Diseño de caja flexible“, y nos aporta una magnífica solución para todos nuestros desarrollos “responsive”. Lo que nos permite es crear un conjunto de elementos flexibles que se adaptan automáticamente a su contenedor y con el que podemos controlar parámetros tales como la alineación, dirección (horizontal/vertical)

Collins, M. J. (2017). Flexbox. In Pro HTML5 with CSS, JavaScript, and Multimedia (pp. 271-285). Apress, Berkeley, CA.

Listas y enlaces

En html se pueden encontrar listas ordenadas, como listas de descripcion. Una lista ordenada comienza con las etiqueta "ol", Cada elemento de la lista comienza con la etiqueta "li" y una lista ordenada comienta con la etiqueta "ul", cada elemento de la lista comienza con la etiqueta "li".

Los enlaces se encuentran en casi todas las paginas web, permiten a los usuarios hacer clic para poder ir desde una pagina a otra un hipervinculo es un texto o una imagen en la que se puede hacer clic y saltar a otro documentos en HTML, enlaces se defienen con- la etiqueta "a"

González, A. H. (2013). Listas dobles y listas múltiples.
